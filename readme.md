##twissues
---
*twissues* is a Github Issues viewer developed by Brian Jacobel for the Twitter Fabric team's coding challenge.

####Resources used
- [generator-gulp-webapp](https://github.com/yeoman/generator-gulp-webapp) (project setup & asset compilation)
- [backbone.js](http://backbonejs.org)
- [underscore.js](http://underscorejs.org) (required by Backbone)
- [jQuery](http://jquery.com) (required by Backbone)
- [backbone-github](https://github.com/opperator/backbone-github) (GitHub API integration library)
